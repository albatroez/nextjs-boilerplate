# Run Locally

## Prerequisites
node >= 20.x

[pnpm](https://pnpm.io/installation) >= 9.x

Install dependencies:

```bash
pnpm install
```
Run the dev server

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
